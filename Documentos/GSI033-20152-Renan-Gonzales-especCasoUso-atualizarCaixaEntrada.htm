Caso de Uso Atualizar Caixa de Entrada
  1. Breve Descrição
    2.

    Este caso de uso trata da comunicação com o sistema de emails, a fim de checar periodicamente se existe algum novo email a ser roteado para os analistas e assistentes.

    O agente deste caso de uso é o próprio sistema.

    2.    Fluxo de Eventos

    O caso de uso é executado periodicamente.

            2.1    Fluxo Básico

O sistema se comunica com o servidor de emails para conferir se existe um email novo. Caso exista, executa o caso de uso Rotear Emails.
2.2    Fluxos Alternativos

          2.2.1    Nenhum novo email no servidor

Caso não haja nenhum email novo no servidor, o caso de uso é encerrado.

    3.    Requisitos Especiais

3.1    Interfaces de Software

O sistema executa com o Microsoft Outlook e utiliza suas APIs de comunicação com o servidor de emails.

     4.    Condições Prévias

Não existem condições prévias associados a este caso de uso.

    5.    Condições Posteriores

Não existem condições posteriores associadas a este caso de uso.

    6.     Pontos de Extensão

Não existem pontos de extensão associados a este caso de uso.
