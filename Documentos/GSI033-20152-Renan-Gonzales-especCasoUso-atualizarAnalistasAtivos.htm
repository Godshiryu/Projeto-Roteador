Caso de Uso Atualizar Analistas Ativos
  1. Breve Descrição
    2.

    Esse caso de uso deve atualizar a lista de analistas disponíveis e alimentar a fila de prioridade de acordo com essa informação.

    O sistema que gerencia os funcionários e os ramais para pausa e ponto é o agente deste caso de uso.

    2.    Fluxo de Eventos

    O caso de uso inicia quando há uma modificação de status de funcionário cujo alerta é disparado pelo sistema principal da empresa.

            2.1    Fluxo Básico

O sistema verifica a lista de funcionários ativos no sistema principal da empresa e atualiza a fila de acordo com a disponibilidade dos analistas e dos assistentes.

2.2    Fluxos Alternativos

Não existem cenários alternativos associados a este caso de uso.

    3.    Requisitos Especiais

3.1    Interface de Comunicação

O sistema deve se comunicar com outros sistemas da empresa, como o sistema que gerencia ponto e pausas dos funcionários.

     4.    Condições Prévias

Não existem condições prévias associados a este caso de uso.

    5.    Condições Posteriores

Não existem condições posteriores associadas a este caso de uso.

    6.     Pontos de Extensão

Não existem pontos de extensão associados a este caso de uso.
