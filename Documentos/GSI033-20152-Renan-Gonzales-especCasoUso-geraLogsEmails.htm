Caso de Uso Gerar Logs de Emails
  1. Breve Descrição
    2.

    Esse caso de uso é responsável por manter um registro de cada movimentação feita com os emails que chegam.

    2.    Fluxo de Eventos

    O caso de uso é executado a cada roteamento de email.

            2.1    Fluxo Básico

Quando o caso de uso Rotear Emails é executado, o caso de uso salva as informações de data e hora de envio e de roteamento do email, remetente e endereço de email do funcionário para o qual o email foi roteado.
2.2    Fluxos Alternativos

Não existem fluxos alternativos associados a este caso de uso.

    3.    Requisitos Especiais

Não existem requisitos especiais associados a este caso de uso.

     4.    Condições Prévias

4.1    Rotear Emails

Quando há alguma movimentação (roteamento) de emails, o caso de uso executa para que o log seja salvo.

    5.    Condições Posteriores

Não existem condições posteriores associadas a este caso de uso.

    6.     Pontos de Extensão

Não existem pontos de extensão associados a este caso de uso.
